Binomial distribution​

Challenge

Pick a number of trials n (start between 5 and 20) and a probability ppp (start between 0.1 and 0.9). Then plot the binomial distribution over the range of integers x=0,1,2,⋯ ,nx = 0, 1, 2, \cdots, nx=0,1,2,⋯,n.

The binomial distribution is given by dbinom() in R, and can be used like this:

dbinom( x, size = n, prob = p )

How does the shape of the binomial differ as you vary nnn and ppp?

Note. The expression for the binomial distribution is:

x∣n,p∼(nx)px(1−p)n−xx|n,p \sim {n \choose x} p^x (1-p)^{n-x}x∣n,p∼(xn​)px(1−p)n−x

Here (nx){n \choose x}(xn​) means 'n choose x' - the number of ways of choosing x things from n things - which can be computed using choose(n,x) in R. For extra kudos, plot this using your own function binomial(x, n, p) implementing the above formula.

Normal distribution​

Challenge

Note. the normal distribution density is given by dnorm() in R, but you have to specify the standard deviation (i.e. the square root of the variance) instead of the variant
